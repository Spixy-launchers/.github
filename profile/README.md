## Что мы делаем

Spixy собирает владельцу Minecraft-сервера брендированный `.exe`-лаунчер с
модами и автообновлением, сайт проекта с донат-магазином, RCON-панель и
аккаунты игроков — всё в браузере, за несколько минут, без кода и без аренды
VPS под раздачу файлов.

Есть бесплатный тариф навсегда: один сервер, сборка до 200 МБ, карта не нужна.

*Spixy is a no-code platform for Minecraft server owners: a branded launcher
`.exe` with mods and over-the-air updates, a project website with a donation
store, an RCON panel and player accounts — built in a browser in minutes.*

## Официальные источники

Spixy распространяется только с этих адресов:

- 🌐 **https://spixy.site** — сайт и дашборд (единственное место, где работает конструктор)
- 📦 **https://github.com/Spixy-launchers/spixy** — основной репозиторий
- 💬 **https://discord.gg/QGbT6HYPs9** — Discord
- ✉️ **spixyworld.on@gmail.com**

Конструктор работает **в браузере** — устанавливать ничего не нужно, Java для
него не требуется. Мы никогда не предлагаем скачать Spixy `.zip`-архивом со
стороннего сайта или из чужого репозитория. Любой другой ресурс, называющий
себя Spixy, к нам отношения не имеет.
